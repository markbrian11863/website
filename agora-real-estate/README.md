# Mark Brian Real Estate Website

A modern, responsive React website for Mark Brian Real Estate, featuring vibrant colors and engaging design to attract more clients.

## Features

- **Modern Design**: Clean, professional layout with smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **Interactive Components**: Hero section with search, featured properties, services showcase
- **Contact Form**: Functional contact form with validation
- **TypeScript**: Built with TypeScript for better development experience
- **Fast Performance**: Built with Vite for optimal performance

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router DOM
- Lucide React (for icons)
- CSS-in-JS with styled-jsx

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Sections

1. **Header** - Navigation with mobile menu
2. **Hero** - Main banner with property search
3. **Featured Properties** - Showcase of premium listings
4. **Services** - Real estate services offered
5. **About** - Company information and achievements
6. **Contact** - Contact form and information
7. **Footer** - Links and additional information

## Customization

The website is designed to be easily customizable:

- Update company information in the components
- Replace placeholder images with actual property photos
- Modify colors and styling in the CSS
- Add more sections or pages as needed

## Deployment

To deploy the website:

1. Build the project:
   ```bash
   npm run build
   ```

2. Deploy the `dist` folder to your hosting provider
